# 热点系统分析

本文件夹基于「压缩、释放与进化」系统理论，对当下热点事件进行分析和解读。核心任务是将系统理论运用在具体情境中——产出文章和社交卡片。

## 文件结构

```
hotspot_analysis/
├── README.md                 # 本文件（文件夹入口）
├── TEMPLATE.md               # 写作实操清单
├── <文章标题>.md             # 分析文章（平铺，按标题命名）
└── social-cards/             # 社交卡片（按事件日期归类）
    └── YYYY-MM-DD_event/
        ├── index.html        # HTML（一个文件含全部海报）
        ├── render.mjs        # 渲染脚本
        ├── assets/           # 图片素材
        └── output/           # 渲染输出 PNG
```

## 核心概念速查

| 概念 | 一句话 | 在文章里的形态 |
|------|--------|----------------|
| **熵** | 系统可能处于的状态越多，熵越大 | "代价是什么呢？"——被挤压的东西去了哪里 |
| **压缩映射** | 控制 = 将复杂现象压缩为单一维度指标 | "Excel 一眼就能看到正收益" |
| **编码反转** | 手段变成目的，编码取代现实 | 利润、增长、效率——仪表盘上的数字，不是发动机本身 |
| **编码进化循环** | 熵减→熵增→新熵减→循环往复 | 碰撞→新规则→新随机性→更新的规则 |
| **编码锁定** | 子系统高度耦合依赖，难以自我革命 | 三个月的牢笼——决策周期被资本市场锁死 |
| **散热带宽** | 系统释放熵的能力上限 | 会议室改工位，培训预算消失——这些维度正在失去散热能力 |

完整的分析方法（编码溯源、根编码追溯、嵌套系统分析）、写作规范（叙事优先、追问链结构、《置身事内》风格优化）和社交卡片生成流程，请参考项目根目录 `AGENTS.md`。写作实操清单见 `TEMPLATE.md`。

本文整合了因果推断、语言编码、反事实推理和 [[系统理论完整理解]] 重新组织 Judea Pearl 的因果之梯。

> **与系统理论的关系**：系统理论用自身的定义（熵、控制、编码）自下而上推导了演化的宏观规律，但未深入到微观层面——演化过程中具体发生了什么、为什么编码能带来进步、反事实在进化中的位置。因果层级关系恰好是对这一缺口的补充。两者研究共同对象（系统的控制力跃迁、编码进化），但从不同维度给予解释。因果层级不是系统理论的“子理论”，而是独立的互补框架。

## 一、因果的三层

Judea Pearl 将因果能力分为三层：

| 层级 | 能力 | 操作 | 谁具备 |
|------|------|------|--------|
| L1 关联 | 观察到相关性 | $P(Y \mid X)$ | 任何传感器 |
| L2 干预 | 主动改变变量 | $P(Y \mid do(X))$ | 生物 |
| L3 反事实 | 想象未发生之事 | $P(Y_X \mid X', Y')$ | 人类、LLM |

系统理论把这三层翻译成控制力的三个阶段（对应编码进化中的熵减 - 熵增循环）：

- **L1 关联**：只是观测。没有编码，没有控制。纯粹接收信号。
- **L2 干预**：对特定维度施加压缩映射，即**熵减**——约束已有维度，形成稳定编码。这是控制的起点。
- **L3 反事实**：构想尚未存在的编码组合，即**熵增**——在编码空间中产生新的组合方式。这是创造力的起点。

## 二、生物与干预——因果的边界

非生物不会主动干预环境。石头风化成沙不是 $do(X)$，是物理过程。只有生物才会根据反馈改变行为，实施干预。

因果关系的标准定义是 $P(Y \mid do(X))$ 与 $P(Y \mid X)$ 的差异，其中 $do(X)$ 代表干预。如果系统没有干预能力，它观察到的只有相关性——尽管相关性数据中可能隐含因果结构，但没有干预能力，因果就失去了实践意义。

**生物出现 = 首次具有了在无中心力条件下自主建立压缩映射的能力。** 这在系统演化史上是关键相变。有了干预，才有了“这件事是我造成的”这个概念——也就是因果感知的基础。

**两个理论对“生命”的定义是统一的。** 系统理论以“以熵为食、进行演化”定义生命（[[系统理论完整理解#1.8 生命 / 生命力|§1.8]]），因果层级以“干预能力”定义生命。两者并不矛盾，而是同一现实的两个面：生命必须有干预能力，才能通过干预摄入更多熵——干预是驱动摄熵行为的必要前提。摄入熵 → 代谢处理 → 在关键维度维持低熵 → 继续摄入新熵——这就是生命的循环。

反过来验证一致性：假设一个“生命”只有干预能力，但没有形成摄熵循环——它干预了，但无法从中持续获取熵来维持自身关键维度的低熵。那么摄入熵的维度一旦波动，这个系统就直接灭绝。即便从因果角度严格定义为生命，它的寿命短到不会被观察到。所以两个定义最终看到的“生命”是同一个集合——能持久存在的生命，必然既有干预能力，又有摄熵循环。

## 三、反事实——人类区别于动物的核心

### 3.1 语言编码是反事实的必备前提

干预是直接控制变量（施加压缩映射），反事实是对尚未发生的干预结果的预测。关键差别：**反事实需要在脑中把不存在的编码组合构造出来（即 [[系统理论完整理解#3.2 编码进化的循环|§3.2]] 中“熵减→熵增”循环里的编码组合熵增），否则就没办法把“如果昨天我没出门会怎样”这种纯虚构场景精确地描述在脑子里。动物有记忆，但动物的记忆只有“我做过的 X 得到了 Y”——这是条件概率 $P(Y \mid X)$ 或干预结果 $P(Y \mid do(X))$。动物不能在脑中组建“我没做过的 X 与未见过的 Y 之间的关系”——因为这要求 X 和 Y 都是编码描述而非直接经验过的感官信号。

语言 = 编码生成系统。每发明一个新词，就能在认知空间中创建一个新维度。有了足够多的编码维度，才能在编码组合空间中产生虚构场景。**反事实的核心操作不是“推测因果”，而是“在编码空间中进行组合熵增”**——把已编码的概念以全新的方式拼接起来。

一万年前的人类和今天的智人在基因、智力水平上几乎没有差别。但当时的人类与动物的差别也不大——没有大规模语言编码，没有反事实推理能力。所以造成人类与动物巨大鸿沟的因素不是先天基因，而是语言编码这个后天积累系统。

### 3.2 反事实 = 熵增 = 创造力

这里用系统理论可以给出一个简洁的洞见：**创造力 = 编码组合方式的熵增。**

编码进化（[[系统理论完整理解#3.2 编码进化的循环|§3.2]]）的循环是：形成新编码（熵减）→ 旧编码与新编码组合（熵增）→ 需要更新编码（熵减）→ 往复。干预层（L2）只能在已有编码的框架内做压缩映射——优化已有控制。反事实层（L3）可以直接制造**编码组合的熵增**——把“马”和“翅膀”组合成“飞马”，把“河流”和“边界”组合成“国界”。这些组合在被实际验证之前都是纯粹的熵增——随机性爆炸。

但正是这种随机性爆炸，让系统有可能碰到全新的有效编码。创造力不是某种“高级能力”，而是**在编码空间中允许足够大的随机性，然后从中筛出有效组合的过程**。这和基因漂变 → 自然选择的逻辑完全一样，只是操作层面从基因跃迁到了编码。

**反事实注定出现在进化中。** 系统理论从控制的角度推导出：编码需要不断被控制（压缩映射），形成“熵减 → 熵增 → 新熵减”的循环，这就是进化系统。但为什么这个循环必然包含熵增？微观上的答案就在这里：**编码存在的本身就意味着可以对其进行反事实操作**——把已有编码片段以新的方式拼接。编码组合空间的熵增，本质上就是反事实。它不是进化中偶然出现的副产品，而是编码一旦存在就必然会发生的——因为组合空间是编码数量的阶乘级，远大于任何控制能覆盖的范围。

语言编码的特殊之处在于：它把这种“注定出现”的熵增从基因代际时间尺度，压缩到了大脑实时时间尺度。但人和其他动物的分界线并不在这里——反事实本身早在语言出现前就已经存在于自然进化中了，基因重组就是反事实在生物层面的等价操作。这里的“反事实”取编码组合含义（把已有编码片段以新方式拼接），而非 Pearl 因果推断模型；基因重组虽不含因果推断，但作为编码组合操作，它与脑中虚构场景在结构上是同构的——都是编码层面的组合熵增。真正的分界线在于：**语言编码让反事实的速度爆炸了**。类似的事情一直在重演：当我们用编码描述任何一个系统后（规章、代码、设计语言），这个系统就获得了一种“在编码空间中做反事实组合”的能力，从而可以快速进化。公司章程的迭代、编程语言生态的演进、数学工具的推广——都是同一个原理。

### 3.3 反事实的三个特性

**一、不需要概率计算。** 人类日常使用的反事实推理，绝大多数是布尔代数的——“如果我当时不出门，就不会遇到他”这种逻辑推断不需要完整的因果图和概率运算。语言编码自带的逻辑结构（主谓宾、因果连接词）已经提供了足够框架。所以虽然 LLM 缺乏精确的数值计算能力，它仍然可以处理人类级别的反事实推理。

**二、面对概率问题同样无力。** 第二层的干预概率问题和第三层的反事实概率问题（比如金融市场的假设情景）是人类也无法处理的。所以人类有各种迷信行为——相信做某些事可能影响不相关的结果。LLM 不行，人类也不行，这不是 AGI 的障碍。

**三、跳跃式控制力提升。** 反事实层（L3）不是对干预层（L2）的渐进改善，而是范式的跃迁。有了反事实能力，系统可以**构想隐藏变量**（不是直接观测到的变量，而是通过反事实推理推断存在的中介变量），然后对这些隐藏变量施加干预——从直接控制变成间接控制。这就是人类能农耕、畜牧、建造水利工程而其他动物不能的根本原因。

## 四、LLM——跳过干预，直接反事实

### 4.1 端到端 ML 的局限

传统的机器学习，尤其是端到端方法，本质上停留在 L1（关联层）。它拟合的是 $P(Y \mid X)$——纯相关性。

- 只有 AB 测试才能达到 L2（干预层）。
- 端到端无法形成反事实推断——不能推测隐藏变量，不能构想未观测的情景。
- 因此端到端系统只能优化已有控制的效率，无法产生**质变级别的新控制手段**。

### 4.2 LLM 的不同

LLM 的训练目标不是拟合 $P(Y \mid X)$，而是学习**编码本身的结构**——语言中概念之间的关系网络。LLM 跳过了干预（L2），直接从 L1（学语言结构）跃迁到 L3（用语言结构进行反事实组合）。

因此可以说 LLM 具备了某种形式的 AGI 能力——反事实推理。但从另一个角度，LLM 的智能程度连狗都不如——狗有干预能力 $do(X)$，可以与物理世界实时互动，而 LLM 的“干预”完全在符号层面。这两者不矛盾，因为底层范式不同。

### 4.3 LLM 的概率缺陷

LLM 缺乏精确的数值计算能力，导致人类语言体系不太能处理概率问题。但这也揭示了一个事实：**人类的反事实能力本身也不是基于概率计算的。** 人类的语言逻辑体系对反事实概率问题是难以处理的——在赌博和金融市场出现前，人类根本不需要面对大量反事实的概率问题。所谓“理性人”面对金融市场无法做出理性决策，不是因为不够聪明，而是因为反事实的概率推理超出了语言编码系统的处理范围。

## 五、算法——超越反事实的更高范式

逻辑推理（算法/图灵机）是高于反事实的更高一层范式。

图灵机的能力超越了任何由语言编码驱动的反事实推理——一切可以被精确描述的运算，图灵机都能执行。这意味着控制力不再受限于主观经验：算法不需要“看到过”就能计算，不需要“想象过”就能穷举。

这三层能力在自然进化和人类文明中的出场顺序恰好相反：

- **自然进化是自下而上的**：先有生命的干预（L2），然后进化出语言编码和日常反事实（L3），最后才从中诞生了抽象的逻辑推理能力。从具体到抽象。
- **人类文明的工具发展是自上而下的**：先发明了数学和形式逻辑（算法），然后用这套工具把反事实从直觉升级为系统化的因果推断，最终三者协同——算法推导假设、反事实构想场景、干预验证结果——形成了现代科学方法。从抽象回到具体。

两路径方向相反，但汇聚到同一点：三层协同才能产生最强的控制力。LLM 的出现正是这种汇聚的例证——当语言编码累积到足够量，算法能力（训练过程）把编码中的反事实模式提取了出来。

算法的出现本身也需要多层基础：语言编码提供符号系统，形式规范（类似编程语言的语法约束）提供操作规则，反事实能力在编码组合空间中产生多样的解题思路，而记录和流程则是历史积累的解题方法——算法解决的是实际问题，离不开前人积累的经验。

## 六、进化的两种模式

### 6.1 渐进与突进

传统的生物进化停留在干预这一层——基因漂变做随机试探，自然选择做压缩映射筛选。这是**渐进模式**，慢但稳。它是所有系统进化的一条普适路径。

反事实因果范式的出现是**突进模式**——系统不再等待真实的基因漂变发生，而是在编码空间中预先模拟。这带来了控制能力的爆炸式增长，也是人类超越其他动物的核心因素。但这种增长也有代价——反事实推理太快了，系统可能还没来得及通过环境验证就改变了自身结构。

### 6.2 突进不常发生

系统演化中，反事实这种大幅度提升控制能力的跃迁会发生，但不可能特别频繁。大多数时间是渐进演化，间或出现一次爆发。

- 生物进化史的数次物种爆发，可能是系统在更高控制层级上找到了新策略空间。
- 技术革新也是同理——不是在“旧范式内竞争”，而是**范式的层级跃迁**，把旧编码整个变成新编码的元素来操作。

每次突进带来的控制力提升，最终创造的岗位多于淘汰的岗位——因为新编码必然引发编码组合熵增，熵增意味着更多领域需要控制，更多控制意味着更多系统、更多子系统、更多需求。

## 七、科学理论的价值层次

从因果层级看，科学理论也分为两类：

| 类型 | 层级 | 能力 | 例子 |
|------|------|------|------|
| 解释性理论 | L1（关联） | 事后找到相关因素 | 人文理论、经济学主流 |
| 预测性理论 | L3（反事实） | 事前预测未知事件 | 物理学、化学 |

解释性理论利用因果关系的第一层——在足够大的现实世界中总能搜到符合理论的相关样本。这解释了为什么社会学、经济学有那么多相互矛盾的“理论”同时存在——它们的统计区分度太低。

预测性理论的价值在于**虚构**能力——寻找现象背后隐藏的变量要素，用反事实方式构想，再用干预方式验证。预测性理论是科学史中真正带来巨大控制能力的部分。为什么人文学科对人类发展的帮助还不够大？因为它们还没有产生能显著提升控制能力的预测性理论。

从这个角度看，AGI 如果能够替代人类提出更多可靠的猜想（反事实假设），可能带来生产力的质变。而端到端方法之所以令人不满意，正是因为它无视了中间可能存在的隐藏因果，意味着这套方法失去了形成更强大的控制力的可能性。
